Listas de definición

Lenguajes de desarrollo web

HTML
Es el lenguaje de marcado que usamos para estructurar y dar significado a nuestro contenido web.
CSS
Es un lenguaje de reglas de estilo que usamos para aplicar estilo a nuestro contenido HTML.
JavaScript
Es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás.
